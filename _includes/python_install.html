Instale o Anaconda

Para instalar o Anaconda, entre no site do Anaconda e faça download da versão individual para o seu sistema operacional (se for Windows, escolha a versão de 64 bits). Você também pode ver como fazer a instalação no Windows no vídeo abaixo. Quando chegar na tela de opções de instalação (1:28 do vídeo), selecione a segunda opção, "Register Anaconda as my default Python 3.7".

Baixe os dados

Baixe os dados nesse link. Faça o unzip na mesma pasta que você está utilizando o Jupyter.

Organize seu espaço de trabalho

Você vai aprender "codando junto" com seu instrutor e colegas. Para fazer isso, você vai precisar tanto da janela que você vai usar para aprender, quanto da janela com a chamada de videoconferência. Recomendamos que você faça uso de alguma dessas opções:

  • Dois monitores: Se você tem dois monitores, Bote o Jupyter ou a janela com a ferramenta de aprendizado em uma tela e a chamada de videoconferência na outra.
  • Dois aparelhos: Se você não tem outro monitor, você pode utilizar outro aparelho com uma tela média/grande, como um celular ou table. No aparelho menor, coloque a videochamada e no seu computador, coloque a ferramenta de aprendizado.
  • Divida sua tela: Se você só tem um aparelho e uma tela, pratique ter duas janelas (a chamada de videoconferência e a ferramenta usada no workshop) abertas juntas. Veja qual é o melhor ajuste nessa configuração, de forma que você consiga trocar facilmente entre as duas.
